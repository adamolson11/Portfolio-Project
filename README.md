# My Portfolio (scaffold)

This repository is a minimal Next.js (App Router) + TypeScript + Tailwind scaffold for a personal portfolio.

Structure:
- app/ — Next.js app routes (layout, page, global styles)
- components/ — UI and layout components
- data/ — projects and social links
- lib/ — types, utils, seo helpers
- public/ — images and static assets
- styles/ — alternate global styles

Getting started:
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Replace placeholder images in `public/images/` and update `data/projects.ts`

Customize the components in `components/` and the content in `data/`.
